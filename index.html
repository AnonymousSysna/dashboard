FAQ

Questions before you request

Is this actually free?

Yes. There is no fee, no trial, and no upsell. I am sharing spare capacity on my existing hosting setup.

How long does setup take?

Usually 24 to 48 hours, depending on my schedule and how complete the request is.

Can I use a custom domain?

Not right now. The offer is limited to subdomains on studentsite.dev and studentsite.online.

What happens if the project outgrows the server?

I will contact you first. If the project needs more resources than the shared server can handle, you will need to move it elsewhere.

Can I move the site later?

Yes. Static and PHP projects can be moved from the files you upload. WordPress sites can be exported using the usual WordPress tools.
